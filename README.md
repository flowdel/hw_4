# De Brujin graph
### Required arguments

* ```-f``` - fasta file
* ```-k``` - kmer size
* ```-v``` - you can choose 'full' or 'nick' view of graph

### Running

Open terminal and run:
```
hw4.py -f reads.fasta -k 20 -v full
```
